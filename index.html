Reglas de Negocio

Define las condiciones, restricciones y lógica que rigen el comportamiento del sistema según el dominio del negocio.

Un usuario no puede registrarse con un correo duplicado.
Un pedido debe tener al menos un producto para ser confirmado.
Un usuario solo puede tener tres préstamos activos simultáneamente.
